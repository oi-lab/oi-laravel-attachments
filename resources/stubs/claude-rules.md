# Laravel File Attachments

Use the `oi-laravel-attachments` package to attach files to any Eloquent model via the `HasAttachments` trait
(named collections, ordering), plus the `File` and `Folder` models and the `StoreUploadedFile` /
`AttachUploadedFiles` upload actions. Resolve model classes through `OiLaravelAttachments` so config overrides apply.

- IMPORTANT: Activate `oilab-laravel-attachments` when working with file uploads, attachments, media, or folders
  in this Laravel application.
